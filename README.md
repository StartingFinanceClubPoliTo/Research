# Area-Research

Research articles and supporting code by **Starting Finance Club PoliTo**.

This repository hosts the codebase, datasets, and reproducibility 
materials for all research papers published by the Research division 
of the Starting Finance Club at Politecnico di Torino.

## Repository Structure

Each published article lives in its own top-level folder, containing:

- A `README.md` describing the article, authors, and the role of each script.
- One subfolder per analytical component (e.g. `production/`, `ebitda_montecarlo/`).
- A `data/` folder when public datasets are committed
